Problem #1: Miscellaneous Functions

  1. Implement the function fizzBuzz, which represents a player for the game fizz buzz. That is, given a positive integer n, the function fizzBuzz returns "fizz" if n is divisible by 3, "buzz" if n is divisible by 5, "fizz buzz" if n is divisible by both 3 and 5, and otherwise the function returns the number n itself. Make sure to return exactly these results; for example, the strings should be all lowercase! Hint: The modulo operation % will come in handy.

  2. Implement a function rps that implements the classic game Rock Paper Scissors. The function takes two arguments, which are respectively the choices of player 1 and player 2. These choices can be assumed to be one of the three strings "rock", "paper", and "scissors". The function should return a string representing the result of the game, which must be one of the three strings "player 1 wins", "player 2 wins", and "tie game". Again, make sure to return exactly these strings!

  3. Implement a function countPos that accepts an integer list l of any length and returns the number of elements of the list that are positive. For example, countPos([1, -4, 0, 4, 8, 0]) returns 3.

  4. Implement a function threshold that takes a list l of numbers and a number n and returns a new list that is identical to l but with all numbers that are less than n replaced by n. For example, threshold([1, -4, 0, 4, 8], 1) returns [1, 1, 1, 4, 8]. As in the previous problem, the given list can be of any length.

  5. Implement a function intRange that takes two integers low and high and returns an ordered list of all integers that are greater than or equal to low and less than high. For example, intRange(2, 5) returns [2,3,4] and intRange(3, 2) returns []. You may not use the built-in range function.

  6. Implement a function innerLists that takes a list l of integers and returns a list of lists, where each element e of the original list l is replaced by the the list [1,...,e]. For example, innerLists([2, 5, 0, 1]) returns [[1,2], [1,2,3,4,5], [], [1]]. Hint: Make use of the intRange function that you defined in the previous problem.
