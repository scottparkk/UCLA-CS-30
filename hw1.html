Problem #1: Python Functions

In this problem you will implement several Python functions. I've provided the hw1.py file to get you started, but there isn't much there! Do not use any language features of Python that have not been covered in class. Do not use any Python libraries other than the math library.
  1. Implement a function totalSeconds that takes 3 integers, in order representing a number of hours, minutes, and seconds. The function should return the total number of seconds that this represents. For example, totalSeconds(1,2,3) evaluates to 3723. Currently the hw1.py file contains an implementation of this function but it just returns 0 -- you should edit it to do the right thing.

  2. Two integers are relatively prime if they have no common factors other than 1. Implement a function relativelyPrime that takes two integers and returns a boolean value indicating if the two numbers are relatively prime. For example, relativelyPrime(14, 15) returns True. Hint: Find a function in Python's math library that you can call to do most of the work.

  3. Implement a function distanceFromSquare that takes a positive integer as an argument and returns its distance from the nearest integer (either above or below it) that is a square. For example, distanceFromSquare(13) returns 3 since 13 is 3 away from 16, which is a square. Hint: Again Python's math library will be useful. Also recall the built-in min function, which returns the minimum of its argument numbers.

  4. Implement a function palindrome4 that takes a string containing exactly four characters and returns a boolean value indicating whether the string is a palindrome (i.e., it is the same forwards as backwards). For example, palindrome4('abba') returns True. Your function should also work on lists that have length 4; for example palindrome4([1, 2, 2, 1]) returns True.

  5. Implement a function palindrome8 that takes a string containing exactly eight characters and returns a boolean value indicating whether the string is a palindrome. Your implementation is not allowed to test characters for equality with one another; instead call your palindrome4 function (multiple times!) to solve this problem.

  6. Optional challenge problem (just for you, no need to turn it in): Implement a function fromTotalSeconds that is in the inverse of the totalSeconds function above. Specifically it takes an integer argument representing a number of seconds and returns a list containing the corresponding number of hours, minutes, and seconds. For example, fromTotalSeconds(3723) evaluates to [1, 2, 3]. Hint: The modulus operator % will be useful, and so will the integer division operator //, which performs division but only returns the integer part of the result.

Problem #2: Picobot

The Picobot playground is here.
  1. Write a Picobot program (i.e., a set of rules) that will cause the robot to traverse every cell in a rectangular world. Your program should work regardless of where the robot starts from in the world, so make sure to test your code with the robot in many different starting places -- you can use the Teleport Robot buttons at the bottom to move the robot to wherever you want it to start. Advice: Write your rules incrementally. That is, write a few rules. See what they do. Then add new rules or modify the rules to try to make them a bit better. Continue in this way until you have a full solution.

    Copy and paste your rules to a text file named pico1.txt. You can use IDLE to create and save this file, or you can use any other text editor that you are comfortable with (note that Microsoft Word is not a text editor, because it does not save its files in plain-text format). Make sure that the file contains only your rules; comments (lines starting with #) are ok, but nothing else should be there. In particular, copying and pasting the entire contents of this file back into the Picobot Rules window should result in a valid solution to the problem. Try doing that and make sure it works!

  2. Press the --> button to the right of the word MAP on the Picobot webpage two times, and you will see a diamond-like world. Write a Picobot program that will cause the robot to traverse every cell in this world, from any starting point. Copy and paste your rules to a text file named pico2.txt. Again, make sure that copying and pasting the entire contents of this file back into the Picobot Rules window results in a valid solution to the problem.

  3. Optional challenge problem (just for you, no need to turn it in): Try to solve the first Picobot problem above in as few rules as possible. How many rules do you need? Same question for the second Picobot problem above.
